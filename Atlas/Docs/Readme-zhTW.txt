==============
= 關於 Atlas =
==============

暴雪公司在遊戲之中內建了艾澤拉斯每一區域的地圖，但基於某些因素，副本地圖並未被
納入。當你在副本時，你依舊可以見到你的周遭地形被顯示在小地圖裡，但是你無法完整
的顯示整個副本。這便是 Atlas 出現的原因。 Atlas 的核心基本上就是一個影像瀏覽器
。然而，這樣的影像也使得 Atlas 顯得頗具價值︰一個在遊戲中可以顯示各個副本的地
圖。更進一步的，地圖上也被標記了各個首領的位置以及其他重要的事物。 Atlas 也包
含了戰場地圖、副本位置圖、飛行地圖、以及野外首領位置圖等。

在安裝好 Atlas 之後，你將會注意到一個新的像是一個小地球的小圖示出現在小地圖的旁
邊，你可以對著小圖示按著右鍵並且在小地圖旁拖曳。對著小圖示按左鍵可以開啟 Atlas 
的主視窗，視窗內顯示地圖與說明。你也可以藉由指令 /atlas 來開啟主視窗。在視窗上
方的下拉選單使你可以瀏覽各種地圖。你也可以移動 Atlas 的主視窗。或是藉由右上方的
上鎖鍵將視窗鎖定位置。

在說明欄的下方有個文字輸入欄位可以讓你搜尋，旁邊上有個重設按鍵。輸入一部份的文字
或是某首領的名稱，並按下搜尋鍵，則說明欄的資訊將被篩選為符合你輸入的文字。要注意
的是搜尋僅限於你當前所開啟的地圖，並不會搜尋整個 Atlas 的資料庫。由於像是怒焰裂
谷這種僅有少數的首領的小副本，搜尋功能變得不是很有用處，因此，搜尋功能將指出現在
具有相當多首領資訊的地方，例如像是黑石深淵，或是 AtlasMajorCities 這個插件。如果
你想要讓整個列表顯示回來，則按下重設按鍵即可。



========
= 設定 =
========

Atlas 只有很少並且很簡單的選項，藉由在主視窗右上方的選項按鈕，或是藉由指令 
/atlas options 可以打開選項設定。某一小部份的設定並不是那麼淺顯易懂，因此這裡將提
供更多關於這些的資訊。「自動選擇副本地圖」功能可以在你打開 Atlas 時自動偵測你所在
的副本，並且自動切換到對應的副本地圖。這功能對於單一副本具有多個地圖的地方將會失
效，例如黑石塔、厄運之槌、或是一些外域副本。在這些狀況之下， Atlas 會記住你最後開
啟的地圖。「點擊滑鼠右鍵開啟世界地圖」功能使你對著 Atlas 主視窗按下右鍵時將可以開
啟世界地圖。這對於你想快速的由 Atlas 的副本地圖切換到世界地圖來說相當有用。「使 
Atlas 不超出畫面」選項使你在移動 Atlas 視窗時，將侷限於遊戲內的範圍。「重設位置」
的功能則相當易懂，當你把 Atlas 移動到你很難在移回來的位置時，便可藉由重設位置來將
視窗位置移回預設位置。



==============
= Atlas 插件 =
==============

Atlas 具有內建的插件系統，這使得額外的地圖可以很輕易的被加入。事實上， Atlas 發
佈時皆包含了五個插件（戰場地圖、副本位置圖、飛行地圖、副本入口、以及野外首領地圖
）。若你對於製作一個自己的 Atlas 地圖套件有興趣，建議你可以從內建的這五個插件之
一著手並當作一種範本。若你需要幫忙，你可以與作者聯繫，或是在 Atlas 的論壇上張貼
訊息，然而我建議你盡量事先多做一些研究。此外，有一些插件比起提供而外的地圖集做的
還要更多。AtlasLoot 提供了相當多的首領掉落寶物表、AtlasQuest 則提供了各個副本的
相關任務訊息。請留意我自己並沒有開發這些插件，他們各有各自的專案負責人。



==============
= 在安裝之前 =
==============

* 任何舊版本的 Atlas 必須事先移除
* 只針對特定版本的 Atlas 所設計的插件（例如 AtlasLoot、AtlasQuest等）必須
  事先刪除



============
= 如何安裝 =
============

* 將套見解壓縮至魔獸世界的目錄
* 你的解壓縮軟體必須能支援目錄結構
* 如果你遇到問題，請造訪 Atlas 的常見問題解答︰
* http://www.atlasmod.com/phpBB3/viewtopic.php?t=3
